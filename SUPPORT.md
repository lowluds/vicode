# Support

If something broke or something feels confusing, open a GitHub Issue in this repo.

Include:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Screenshots or a screen recording if possible
- Browser/device info

## Feature requests

Open an Issue labeled `feature request`.

Keep it concrete:
- Problem: what's missing or broken?
- Who is it for?
- What does "done" look like?
- What's the smallest useful version?
